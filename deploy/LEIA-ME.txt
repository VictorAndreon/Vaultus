============================================
   VAULTUS - COMO USAR
============================================

PASSO 0 - Baixar o Vaultus (so na primeira vez)
-----------------------------------------------
1. Instale o Git para Windows (pode aceitar as opcoes padrao):
      https://git-scm.com/download/win
2. Abra o "Prompt de Comando" (menu Iniciar, digite "cmd") e cole:
      git clone https://github.com/VictorAndreon/Vaultus.git "%USERPROFILE%\Vaultus"
3. Pronto: a pasta do Vaultus fica em C:\Users\SEU-USUARIO\Vaultus
   e os instaladores ficam na subpasta "deploy" dentro dela.


PASSO 1 - Instalar o Docker Desktop (so na primeira vez)
--------------------------------------------------------
Baixe e instale o Docker Desktop:
   https://www.docker.com/products/docker-desktop/
Se ele pedir para instalar/atualizar o WSL 2 ou reiniciar o
computador, aceite e reinicie.
Depois de instalar, ABRA o Docker Desktop pelo menu Iniciar e
espere ele terminar de carregar (o icone da baleia fica parado).


PASSO 2 - Instalar o Vaultus (so na primeira vez)
-------------------------------------------------
Abra a pasta Vaultus\deploy e de um duplo-clique em:
   "Instalar Vaultus"
Uma janela preta vai aparecer mostrando o progresso.
A PRIMEIRA vez DEMORA VARIOS MINUTOS (ele baixa e prepara tudo).
Nao feche a janela ate aparecer "Pronto!".

Ao final, o navegador abre sozinho na tela de criar conta
(http://localhost/register) e tres atalhos sao criados na
Area de Trabalho:
   - "Vaultus"            -> abre o app
   - "Parar Vaultus"      -> desliga o app
   - "Atualizar Vaultus"  -> atualiza para a versao mais nova

Na PRIMEIRA tela, CRIE SUA CONTA (seu nome, email e senha).
Esse cadastro so funciona uma vez: a conta que voce criar e a sua.
Guarde bem o email e a senha que escolher.


PASSO 3 - No dia a dia
----------------------
Basta o atalho "Vaultus" na Area de Trabalho.
Ele liga tudo sozinho e abre o navegador.


LOGIN
-----
   Endereco: http://localhost
   Email/Senha: os que VOCE escolheu ao criar a conta no primeiro acesso.
   (Nao existe senha padrao - a conta e sua.)


ATUALIZAR
---------
Use o atalho "Atualizar Vaultus". Ele baixa a versao nova
SEM apagar seus dados (login e banco continuam).


SE ALGO DER ERRADO
------------------
1. Confira se o Docker Desktop esta ABERTO.
2. Rode "Instalar Vaultus" de novo (e seguro repetir).
3. Se ainda assim falhar, avise quem cuida do Vaultus.
